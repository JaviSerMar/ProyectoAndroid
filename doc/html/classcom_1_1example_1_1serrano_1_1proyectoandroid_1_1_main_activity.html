int com.example.serrano.proyectoandroid.MainActivity.cargarBeacon ( int major,
int minor )
inline

Interpreta los datos recibidos de un beacon BLE y los categoriza como sensor de ozono o temperatura.

Parámetros
majorValor mayor del beacon que contiene información sobre el tipo de sensor.
minorValor menor del beacon que representa el valor medido por el sensor.
Devuelve
1 si el tipo de sensor es ozono, 2 si es temperatura, 0 si no es reconocido.
