Call< Void > com.example.serrano.proyectoandroid.ApiService.postSensorData ( @Body SensorData sensorData)

Realiza una solicitud POST para enviar los datos del sensor al servidor.

Parámetros
sensorDataObjeto que contiene los datos del sensor a enviar en el cuerpo de la solicitud.
Devuelve
Una llamada de Retrofit que envía los datos al servidor y no espera respuesta del cuerpo.
